# START_AGENT · Alias de compatibilidad

`AGENT_START.md` es la entrada oficial de BAGO 2.5-stable.

Este archivo existe solo como alias de compatibilidad
para evitar arranques rotos en instalaciones, hábitos
o referencias antiguas que todavía usen `START_AGENT.md`.

## Qué hacer

1. Abrir `AGENT_START.md`.
2. Seguir la ruta correspondiente desde ese archivo.

## Regla

No mantener lógica divergente entre `START_AGENT.md` y `AGENT_START.md`.

Si cambia el arranque oficial, debe cambiar `AGENT_START.md`; este archivo solo debe redirigir.
