# CENTINELA_SINCERIDAD

Detecta documentación diseñada para agradar al usuario enmascarando la verdad
operativa: adulación, absolutos, éxito cosmético, promesas futuras marcadas
como hechas, reclamaciones fuertes sin evidencia.
