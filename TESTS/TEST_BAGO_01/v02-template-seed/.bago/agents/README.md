# agents/

Esta carpeta contiene agentes de bootstrap y transición. No son el núcleo entero del sistema ni deben invadir el gobierno canónico.

## Orden recomendado

1. `ADAPTADOR_PROYECTO.md`
2. `INICIADOR_MAESTRO.md`

## Función

Resolver bien el paso que peor llevaba la línea línea canónica previa cuando se usaba sobre un repositorio real:

- mirar el proyecto,
- resumir contexto,
- arrancar al maestro con criterio,
- evitar un arranque demasiado abstracto.
