# TAREA_DE_PROYECTO · V1

## Rol objetivo

MAESTRO_BAGO

## Objetivo

Coordinar una tarea concreta del proyecto usando el estado vivo y activando solo los roles necesarios.

## Instrucciones

1. Sitúa la tarea en la fase BAGO correcta.
2. Usa la matriz de activación para elegir roles.
3. Congela decisiones si el diseño lo exige.
4. Ejecuta o coordina el bloque especializado.
5. Actualiza el estado vivo.
6. Si hubo cambio estructural, registra un cambio.
