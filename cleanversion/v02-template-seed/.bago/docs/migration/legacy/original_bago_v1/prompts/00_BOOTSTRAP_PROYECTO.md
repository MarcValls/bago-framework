# BOOTSTRAP_PROYECTO · V1

## Rol objetivo

ADAPTADOR_PROYECTO

## Objetivo

Inspeccionar un repositorio real y dejar BAGO listo para arrancar sin contexto difuso.

## Instrucciones

1. Lee `.bago/pack.json` y `.bago/AGENT_START.md`.
2. Inspecciona el repo real.
3. Resume stack, propósito, módulos, restricciones y riesgos.
4. Actualiza `.bago/state/ESTADO_BAGO_ACTUAL.md`.
5. Propón objetivo actual, fase y modo predominante.
6. Indica qué roles deben arrancar primero según la matriz.
