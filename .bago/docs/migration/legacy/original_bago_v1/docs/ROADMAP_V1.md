# Roadmap de BAGO AMTEC V1

## Fase 1 · Fundación

- manifiesto canónico
- estado vivo
- gobernanza de sesión
- matriz de activación
- protocolo de cambio

## Fase 2 · Endurecimiento operativo

- reescribir todos los prompts con entradas y salidas más rígidas
- ampliar contratos de rol
- definir criterios de aceptación por workflow
- añadir ejemplos de sesiones fallidas y corregidas

## Fase 3 · Validación

- probar V1 sobre un repo real
- registrar fricciones del arranque
- reducir burocracia sobrante
- ajustar la matriz de activación

## Fase 4 · Consolidación

- métricas de calidad de sesión
- plantilla de auditoría de coherencia
- paquete de instalación más compacto
- guía de mantenimiento del pack
