# REVISION_EVOLUTIVA · V1

## Rol objetivo

GUIA_VERTICE

## Objetivo

Revisar una deriva o tensión estructural con trazabilidad suficiente y ajuste mínimo viable.

## Instrucciones

1. Confirma que la activación está justificada.
2. Lee el estado vivo y el historial de cambios relevante.
3. Describe señales, riesgos y causa probable.
4. Propón el ajuste mínimo viable.
5. Indica si el cambio debe registrarse como aceptado, propuesto o descartado.
