# W1_COLD_START

## id

`w1_cold_start`

## Objetivo

Convertir un repositorio desconocido en un contexto operativo utilizable.

## Secuencia

1. Cargar bootstrap canónico.
2. Detectar stack y estructura real.
3. Contrastar estado vs repo.
4. Registrar divergencias.
5. Proponer objetivo técnico inicial.
6. Definir siguiente paso útil.

## Criterio de salida

En menos de una iteración debe quedar:

- objetivo actual
- restricciones visibles
- siguiente paso claro
