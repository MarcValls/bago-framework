📖 Guía rápida BAGO (2 minutos)

  1. Primera vez: Ejecuta ./bago setup en terminal
  2. Ver ideas: ./bago ideas lista tareas priorizadas
  3. Trabajar: ./bago session abre contexto de trabajo
  4. Estado: ./bago status o este panel web

🎯 Flujo de trabajo recomendado

./bago ideas                # 1. Lista ideas
./bago ideas --accept 3     # 2. Acepta idea #3
./bago session              # 3. Abre terminal con contexto
# [haces cambios]
./bago task --done          # 4. Marca completada

❓ Preguntas frecuentes

¿Qué es BAGO?
Sistema de gobernanza de sesiones IA para mantener contexto, trazabilidad y workflows estructurados.
¿Cómo inicio el servidor?
python3 .bago/tools/bago_chat_server.py
¿Funciona sin servidor?
Sí, este panel funciona sin servidor. El servidor solo añade datos en vivo.
