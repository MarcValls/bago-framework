# CAMBIO BAGO

- **Cambio ID:** BAGO-CHG-001
- **Fecha:** 2026-04-09
- **Motivo:** Necesidad de evolucionar V.0 a una base más gobernable y trazable.
- **Ámbito afectado:** manifiesto, estado, arranque, activación de roles y protocolo de cambio.
- **Situación previa:** V.0 funcionaba como kit documental operativo pero sin manifiesto canónico ni separación fuerte entre modelo y estado vivo.
- **Cambio aplicado o aceptado:** creación de BAGO AMTEC V1 base con `pack.json`, estado vivo en `state/`, matriz de activación, gobernanza y protocolo de trazabilidad.
- **Impacto esperado:** arranque más consistente, menos ambigüedad entre sesiones y mejor capacidad de revisión.
- **Riesgos asociados:** incremento documental inicial y necesidad de validación práctica en repositorio real.
- **Estado:** aceptado
