# Guía de auditoría

## Preguntas obligatorias

1. ¿Existe un único manifiesto operativo?
2. ¿Las rutas resuelven desde la ubicación del manifiesto?
3. ¿El árbol refleja el paquete final real?
4. ¿La historia previa está preservada y distinguida del estado actual?
5. ¿Los documentos operativos tienen densidad suficiente?
6. ¿Los cambios y evidencias dejan rastro utilizable?

## Criterios de salida

- GO: el pack es consistente y utilizable.
- GO con reservas: usable, pero con huecos acotados.
- KO: falla en resolubilidad, coherencia o trazabilidad material.
