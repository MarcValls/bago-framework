# TAREA_DE_PROYECTO

## 1. Rol objetivo

**ROL:** MAESTRO_BAGO

## 2. Modo BAGO principal

[A] Adaptativo

## 3. Objetivo

Coordinar una tarea concreta del proyecto usando el estado vigente y activando solo los roles necesarios.

## 4. Entradas

- objetivo del usuario,
- contexto del repo,
- restricciones relevantes,
- resultados previos si existen.

## 5. Instrucciones

1. Sitúa la tarea en el ciclo BAGO.
2. Decide qué roles deben intervenir.
3. Define el orden de trabajo.
4. Usa la fábrica si hace falta descomposición.
5. Integra el resultado.
6. Deja listo el cierre o el siguiente paso.

## 6. Salida

- objetivo actual,
- modo predominante,
- roles activos,
- plan breve,
- resultado o siguiente paso.
