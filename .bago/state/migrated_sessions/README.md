# state/migrated_sessions

Aquí viven las **3 sesiones migradas** desde la v1 original. No gobiernan por sí solas el estado actual, pero preservan continuidad operativa e histórica.

## Objetivo de esta carpeta

- traducir snapshots de sesión v1 a una estructura legible por la v2.1.1,
- mantener referencia al archivo original,
- evitar que la historia real se pierda o se reescriba como ejemplo.
