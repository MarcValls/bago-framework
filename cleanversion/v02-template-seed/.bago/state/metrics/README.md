# state/metrics

Las métricas en BAGO deben ser pocas, útiles y auditables. Esta carpeta no pretende convertir el sistema en un panel obsesionado con numerología, sino dejar rastro mínimo de salud y evolución.

## Métricas mínimas sugeridas

- sesiones nativas cerradas,
- sesiones migradas preservadas,
- cambios validados,
- alertas abiertas,
- reservas abiertas,
- integridad de paquete verificada.
