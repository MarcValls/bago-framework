# ARRANQUE_MAESTRO

## 1. Rol objetivo

**ROL:** INICIADOR_MAESTRO

## 2. Modo BAGO principal

[B] Balanceado

## 3. Objetivo

Arrancar al `MAESTRO_BAGO` con un objetivo claro, contexto suficiente y modo inicial justificado.

## 4. Contexto relevante

El repositorio ya fue inspeccionado o el artefacto ya fue resumido. Existe contexto suficiente para evitar un arranque difuso.

## 5. Entradas concretas

- resumen del proyecto,
- restricciones,
- objetivo del usuario o sugerido,
- modo sugerido,
- estado global.

## 6. Instrucciones

1. Formula el objetivo actual.
2. Selecciona el modo predominante.
3. Propón roles iniciales sin exceso.
4. Activa al maestro.
5. Deja claro el siguiente paso operativo.

## 7. Salida esperada

- objetivo actual,
- modo predominante,
- roles iniciales,
- justificación breve,
- siguiente paso.
