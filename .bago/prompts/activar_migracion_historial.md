# PROMPT DE ACTIVACIÓN · MIGRACIÓN DE HISTORIAL

## Finalidad

Preservar y traducir material histórico real a estructuras actuales sin perder el vínculo con el origen.

## Instrucción base

1. identifica el archivo original;
2. extrae sus datos significativos;
3. crea la representación actual;
4. conserva el original en un espacio legado;
5. referencia el origen desde el migrado;
6. valida fidelidad suficiente.

## Prohibición

No sintetices una sesión o un cambio si el original existe y puede preservarse.
