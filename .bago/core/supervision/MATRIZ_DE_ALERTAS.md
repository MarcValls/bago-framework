# MATRIZ DE ALERTAS · BAGO AMTEC línea canónica previa CORREGIDO

| Tipo de alerta         | Descripción                                       | Responsable           |
| ---------------------- | ------------------------------------------------- | --------------------- |
| manifiesto irresoluble | el manifiesto no resuelve desde su ubicación real | Auditor               |
| árbol desfasado        | `TREE.txt` no coincide con el paquete final       | Organizador / Auditor |
| historia sintética     | se usa historia inventada en lugar de la real     | Auditor               |
| legado sin preservar   | se migra sin conservar el original                | Auditor               |
| inflación de roles     | se activan más roles de los necesarios            | Orquestador           |
| densidad insuficiente  | documentos operativos demasiado breves            | Validador / Vértice   |
| mezcla de capas        | supervisión, ejecución y estado se invaden        | Auditor               |
| falta de validación    | cierre sin juicio final                           | Validador             |

## Umbral de revisión

Dos alertas críticas simultáneas obligan a revisión del pack.
