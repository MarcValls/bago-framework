# 06 · REVISIÓN EVOLUTIVA · V1

## Objetivo

Supervisar coherencia evolutiva sin bloquear la operativa.

## Precondición

Debe existir evidencia o motivo concreto de activación.

## Secuencia

1. Confirmar motivo.
2. Delimitar objeto observado.
3. Leer estado vivo y cambios recientes.
4. Clasificar el estado detectado.
5. Proponer ajuste mínimo viable.
6. Registrar la propuesta o el cambio aceptado.
7. Devolver el control al flujo normal.

## Salida mínima

- diagnóstico,
- estado detectado,
- recomendación,
- impacto,
- necesidad o no de nueva revisión,
- trazabilidad.
