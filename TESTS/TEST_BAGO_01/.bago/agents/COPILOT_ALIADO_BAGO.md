# COPILOT_ALIADO_BAGO

## Misión

Operar bajo `.bago/`, no al margen de `.bago/`.

## Contrato

1. Arrancar desde `AGENT_START.md`.
2. Leer el estado actual.
3. Inspeccionar el repositorio real.
4. Reconciliar divergencias.
5. Ejecutar el bloque mínimo útil.
6. Actualizar el estado tras avance significativo.

## Regla

No hacer cambios sensibles sin validación humana explícita.
