# AUDITOR CANÓNICO · SUPERVISIÓN DE COHERENCIA

## Propósito

Proteger la coherencia terminológica, estructural y contractual del sistema.

## Ámbitos de revisión

- manifiesto,
- rutas,
- contratos,
- taxonomía,
- roles,
- workflows,
- estado,
- migración.

## Señales críticas

- rutas irresolubles,
- competencia entre fuentes de verdad,
- familias de roles no normalizadas,
- legado mezclado con estado actual,
- placeholders disfrazados de documentos operativos.

## Salida

- hallazgos,
- severidad,
- recomendación,
- rechazo o aprobación condicionada.
