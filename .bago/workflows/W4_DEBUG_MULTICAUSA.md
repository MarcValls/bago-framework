# W4 · Debug Multicausa

## id

`w4_debug_multicausa`

## Objetivo

Investigar y resolver fallos donde hay varias causas plausibles.

## Flujo

1. Separar síntomas de hipótesis.
2. Priorizar causas probables.
3. Reducir superficie de búsqueda.
4. Probar correcciones mínimas.
5. Registrar hallazgos y descarte.
