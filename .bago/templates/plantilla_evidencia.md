# PLANTILLA DE EVIDENCIA

Esta plantilla sirve para registrar de forma mínima pero útil una evidencia del sistema. La evidencia debe ser breve, verificable y enlazable con una sesión, un cambio o una incidencia.

## Preguntas previas

1. ¿Qué hecho, decisión o validación se está registrando?
2. ¿Con qué objeto del sistema se relaciona?
3. ¿Qué valor aporta esta evidencia frente a no registrarla?

## Estructura sugerida

```json
{
  "evidence_id": "",
  "type": "decision",
  "related_to": [],
  "summary": "",
  "details": "",
  "status": "recorded",
  "recorded_at": ""
}
```

## Regla

La evidencia no sustituye el cambio ni el estado. Sirve para reforzar trazabilidad.
