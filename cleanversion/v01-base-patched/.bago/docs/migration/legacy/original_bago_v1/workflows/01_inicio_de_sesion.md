# 01 · INICIO DE SESIÓN · V1

## Objetivo

Abrir una sesión con contexto real, estado vivo actualizado y selección mínima de roles.

## Secuencia

1. Leer `pack.json` y `AGENT_START.md`.
2. Leer el estado vivo.
3. Inspeccionar el repo real.
4. Actualizar el estado vivo.
5. Activar `MAESTRO_BAGO`.
6. Declarar objetivo, fase y roles.
7. Empezar la tarea.

## Salida mínima

- resumen del proyecto,
- estado vivo actualizado,
- objetivo actual,
- modo predominante,
- siguiente paso.
