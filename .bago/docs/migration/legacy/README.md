# Legacy preservado

Esta carpeta guarda la copia preservada de la `.bago` original v1 que sirvió de base para la migración. Su función no es gobernar el sistema actual, sino permitir:

- auditoría retrospectiva,
- verificación de fidelidad de migración,
- consulta semántica cuando el JSON migrado no capture todos los matices del markdown original.

## Nota de lectura

Los documentos preservados aquí usan el vocabulario de su época. Términos como "pendiente", "activa" o "en curso" describen el estado histórico de esa versión y no deben interpretarse como estado operativo actual del pack BAGO v2.5-stable.
