# BAGO — Directorio de Informes

> Versión: 2.5-stable

## Propósito

Este directorio centraliza los informes generados sobre el estado, la coherencia y la evolución del framework BAGO.

## Tipos de informe

| Tipo | Prefijo | Descripción |
|------|---------|-------------|
| Auditoría de coherencia | `auditoria_` | Revisión de consistencia entre docs, código y estado |
| Informe de sesión | `sesion_` | Resumen de trabajo realizado en una sesión operativa |
| Informe de release | `release_` | Estado del sistema en el momento de una release |
| Análisis de evolución | `evolucion_` | Comparativa entre versiones del framework |

## Convención de nombres

```
YYYY-MM-DD_<tipo>_<descripcion_corta>.md
```

Ejemplos:
```
2026-04-20_auditoria_coherencia_v2.5-stable.md
2026-04-20_release_v2.5-stable.md
```
